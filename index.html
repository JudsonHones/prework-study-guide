Prework Study Guide

HTML

  • the head element contains information about the webpage.
  • the body element represents the visible content shown to the user.
  • ul and ol functions create unordered and ordered lists respectively
  • li creates list items within the ul and ol functions
  • img contains image info
  • src defines the source of an image
  • br creates a line break
  • alt function is a string used to describe the appearance and functionality of an image.
  • p is paragraph information

Git

  • git must precede every git statement
  • git checkout command is used to move branches
  • -b creates a new branch
  • add command adds modifications in the current branch to the staging area.
  • -a flag adds all changes
  • git commit -m" "commits changes with the content in -m being the change note
  • pull command receives modifications into the local environment. **ALWAYS PULL BEFORE WORKING**
  • push command sends changes made locally to the remote branch
  • git status is used to verify the current branch
